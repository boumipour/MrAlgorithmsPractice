یک رشته‌ی اعداد رومی را به مقدار صحیح آن تبدیل کن. اعداد رومی از هفت نماد استفاده می‌کنند و
یک قانون تفریق دارند: وقتی نماد کوچک‌تری پیش از نماد بزرگ‌تر بیاید، تفریق می‌شود نه جمع.

### نمادهای اعداد رومی

| نماد | مقدار |
| --- | --- |
| I | ۱ |
| V | ۵ |
| X | ۱۰ |
| L | ۵۰ |
| C | ۱۰۰ |
| D | ۵۰۰ |
| M | ۱۰۰۰ |

شش حالت تفریقی عبارتند از: **IV** = ۴، **IX** = ۹، **XL** = ۴۰، **XC** = ۹۰، **CD** = ۴۰۰، **CM** = ۹۰۰.

### رویکرد یک‌گذری

رشته را از چپ به راست با جستجوی دیکشنری پیمایش می‌کنیم:

- اگر نماد فعلی **کمتر از** نماد بعدی باشد → بخشی از یک جفت تفریقی است، پس آن را **کم کن**.
- در غیر این صورت → آن را **اضافه کن**.

```
"MCMXCIV"
 M  → بعدی C است (1000 > 100)  → جمع    1000  → 1000
 C  → بعدی M است (100 < 1000)  → تفریق   100  →  900
 M  → بعدی X است (1000 > 10)   → جمع    1000  → 1900
 X  → بعدی C است (10 < 100)    → تفریق    10  → 1890
 C  → بعدی I است (100 > 1)     → جمع     100  → 1990
 I  → بعدی V است (1 < 5)       → تفریق     1  → 1989
 V  → بعدی ندارد               → جمع       5  → 1994
```

### پیچیدگی

| | زمان | حافظه |
| --- | --- | --- |
| یک گذر | O(n) | O(1) |

دیکشنری دقیقاً ۷ ورودی ثابت دارد، بنابراین حافظه عملاً ثابت است.

### توضیح برای یک بچه 🧒

تصور کن داری یک برچسب قیمت را می‌خوانی که به یک کد مخفی نوشته شده. هر حرف یک قیمت دارد.
قیمت‌ها را با هم جمع می‌زنی — اما یک ترفند وجود دارد: اگر یک حرف ارزان‌تر **درست قبل از** یک حرف گران‌تر بیاید،
آن را **کم می‌کنی** نه اینکه اضافه کنی.

پس `IV` یعنی «۵ منهای ۱» = **۴**، و `IX` یعنی «۱۰ منهای ۱» = **۹**.
